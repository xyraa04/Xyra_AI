## 🌌 Overview

**XYRA AI** is an AI-powered developer tool that helps you **understand, debug, and design codebases in seconds**.

Instead of spending hours reading unfamiliar code, XYRA AI gives you:

- 📖 Clear project explanations  
- 🗂️ Smart file breakdowns  
- ⚡ Instant onboarding guidance  
- 🧩 Architecture insights
